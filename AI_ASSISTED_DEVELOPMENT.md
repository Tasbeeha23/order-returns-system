# AI Assisted Development

## Overview

This project was developed using AI-assisted development tools as part of the implementation process. AI was used to accelerate development, improve productivity, and assist with documentation, while all design decisions, validation, testing, and final reviews were performed manually.

---

## AI Tools Used

* Claude (Anthropic)
* ChatGPT

---

## How AI Assisted Development

### Architecture and Planning

AI was used to help break down the requirements into a structured full-stack application consisting of:

* React frontend
* Spring Boot backend
* H2 in-memory database
* REST API layer
* Business rule evaluation component

It also assisted in defining the folder structure, API design, and overall application flow.

### Code Generation and Scaffolding

AI helped generate boilerplate and repetitive code such as:

* DTOs
* Entity classes
* Repository interfaces
* Controller endpoints
* Service layer structure
* React components
* Validation logic

This accelerated implementation and allowed more time to focus on business requirements and testing.

### Documentation

AI assisted in drafting project documentation, including:

* README documentation
* API documentation
* Setup and run instructions

All generated documentation was reviewed and refined before inclusion in the final submission.

---

## Human Contributions

The following activities were performed through manual review and decision-making:

* Interpreting project requirements
* Defining return eligibility rules
* Selecting the technology stack
* Reviewing generated code
* Verifying frontend-backend integration
* Running and testing the application locally
* Debugging configuration and environment issues
* Validating business rule behavior and expected outcomes

---

## Challenges Encountered

Some of the key challenges during development included:

* Maintaining consistency between frontend and backend data models
* Ensuring API endpoints matched frontend expectations
* Validating business rule behavior for different return scenarios
* Configuring and testing the complete application locally

These challenges were addressed through iterative testing, code review, and refinement.

---

## Outcome

AI-assisted development significantly accelerated implementation while allowing focus on problem-solving, validation, and overall application quality. The final solution was reviewed, executed, and tested end-to-end to ensure that all functional requirements were met.
